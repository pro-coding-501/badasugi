개인정보처리방침

시행일자: 2026년 1월 12일

받아쓰기(이하 "회사")는 개인정보 보호법 등 관련 법령을 준수하며, 이용자의 개인정보를 보호하기 위해 다음과 같은 개인정보처리방침을 수립·공개합니다.

본 개인정보처리방침은 회사가 제공하는 음성 인식 기반 텍스트 변환 서비스(이하 "서비스")에 적용됩니다.

제1조 (개인정보의 처리 원칙)

회사는 서비스 제공에 필요한 최소한의 정보만을 처리하며, 이용자의 개인정보를 원칙적으로 수집·저장하지 않습니다.

제2조 (처리하는 개인정보 항목)

회사는 다음과 같은 개인정보를 처리할 수 있습니다.

1. 필수 수집 항목: 없음

본 서비스는 회원가입 및 로그인을 요구하지 않으며, 이용자의 개인정보를 직접 수집하지 않습니다.

2. 결제 및 고객지원 과정에서 수집될 수 있는 정보

  • 환불 또는 문의 대응 과정에서 이용자가 자발적으로 제공하는 정보
    (이메일 주소, 결제 내역 확인 정보 등)

제3조 (음성 데이터 및 콘텐츠 처리)

  1. 본 서비스는 원칙적으로 이용자의 음성 데이터를 외부 서버로 전송하거나 저장하지 않습니다.
  2. 모든 음성 인식 및 텍스트 변환 작업은 이용자의 기기 내에서 로컬로 수행됩니다.
  3. 로컬에서 처리된 음성 데이터는 텍스트 변환 즉시 파기되며, 회사는 이를 별도로 저장하거나 기록하지 않습니다.
  4. 회사는 이용자의 음성 데이터 또는 변환된 텍스트를 인공지능 학습, 분석, 마케팅 등의 목적으로 사용하지 않습니다.

제4조 (외부 인공지능 서비스 이용 시)

이용자가 선택적으로 클라우드 기반 인공지능 기능을 활성화하는 경우, 음성 데이터가 외부 API 제공업체로 전송될 수 있습니다.

  1. 서비스에서 지원하는 외부 API는 다음과 같습니다:
    • OpenAI (Whisper API)
    • Deepgram
    • ElevenLabs
    • Google Gemini
    • Groq
    • Mistral AI
    • Soniox
    • 기타 OpenAI 호환 API
  2. 이용자가 외부 API를 사용하는 경우, 해당 API 제공업체의 개인정보처리방침 및 약관이 적용되며, 이용자는 각 API 제공업체의 정책을 확인하고 동의한 것으로 간주됩니다.
  3. 외부 API 사용 시 음성 데이터는 이용자가 직접 입력한 API 키를 통해 암호화되어 전송되며, 회사는 해당 데이터를 저장하거나 별도로 처리하지 않습니다.
  4. 회사는 외부 인공지능 서비스 제공업체의 데이터 처리 행위에 대해 책임을 지지 않으며, 이용자는 각 API 제공업체의 정책을 확인하고 사용할 책임이 있습니다.

제5조 (개인정보의 처리 목적)

회사는 다음의 목적에 한하여 개인정보를 처리할 수 있습니다.

  1. 결제 확인 및 환불 처리
  2. 고객 문의 대응 및 기술 지원
  3. 법령에 따른 의무 이행

제6조 (개인정보의 보유 및 이용 기간)

  1. 회사는 개인정보를 원칙적으로 저장하지 않습니다.
  2. 다만, 법령에 따라 보존이 필요한 경우에 한해 해당 법령에서 정한 기간 동안 개인정보를 보관합니다.
    • 전자상거래 등에서의 소비자 보호에 관한 법률에 따른 보관
      • 계약 또는 청약철회 기록: 5년
      • 대금결제 및 재화 공급 기록: 5년
      • 소비자 불만 또는 분쟁 처리 기록: 3년

제7조 (개인정보의 제3자 제공)

회사는 이용자의 개인정보를 제3자에게 제공하지 않습니다.

단, 법령에 따라 요구되는 경우는 예외로 합니다.

제8조 (개인정보 처리의 위탁)

회사는 원칙적으로 개인정보 처리 업무를 외부에 위탁하지 않습니다.

다만, 결제 처리 등 서비스 운영을 위해 불가피하게 위탁이 필요한 경우, 관련 법령에 따라 다음과 같이 공개합니다.

결제 처리 위탁

  • 위탁 업체: [결제 대행사명 입력 필요]
  • 위탁 업무 내용: 결제 처리, 결제 정보 확인
  • 위탁 기간: 서비스 제공 기간 동안
  • 위탁 정보 항목: 결제 정보(카드번호, 계좌정보 등은 위탁 업체에서 직접 처리하며 회사는 접근하지 않음)

※ 결제 대행사 정보는 실제 사용하는 결제 서비스에 맞게 수정이 필요합니다.

제9조 (이용자의 권리)

이용자는 관련 법령에 따라 다음과 같은 권리를 가집니다.

  1. 개인정보 열람 요청
  2. 개인정보 정정·삭제 요청
  3. 개인정보 처리 정지 요청

관련 요청은 고객센터를 통해 접수할 수 있으며, 회사는 지체 없이 조치합니다.

제10조 (개인정보 보호를 위한 안전조치)

회사는 개인정보 보호를 위해 다음과 같은 조치를 취하고 있습니다.

  1. 개인정보 최소 처리 원칙 준수
  2. 내부 접근 권한 제한
  3. 보안 취약점 관리 및 점검

제11조 (개인정보 보호책임자)

제12조 (개인정보처리방침의 변경)
