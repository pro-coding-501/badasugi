# 토스페이먼츠 실제 서비스 전환 가이드

## 현재 상태
- ✅ 테스트 모드로 설정되어 있음
- ✅ 실제 결제는 발생하지 않음
- ✅ 테스트 카드로 결제 플로우 확인 가능

## 실제 서비스로 전환하는 방법

### 1단계: 토스페이먼츠 사업자 인증 및 라이브 키 발급

1. [토스페이먼츠 개발자센터](https://developers.tosspayments.com/) 접속
2. 로그인 후 **"내 서비스"** 메뉴로 이동
3. **"사업자 인증"** 완료 (사업자등록번호 필요)
4. **"인증키 관리"** 메뉴에서 **라이브 키** 발급
   - 클라이언트 키: `live_ck_...` (웹사이트용)
   - 시크릿 키: `live_sk_...` (서버용)

### 2단계: 웹사이트 키 변경

**파일:** `Website/index.html`

```javascript
// 1358번째 줄 근처
// 변경 전 (테스트)
const TOSS_CLIENT_KEY = 'test_ck_D5GePWvyJnrK0W0k6q8gLzN97Eoq';

// 변경 후 (라이브)
const TOSS_CLIENT_KEY = 'live_ck_발급받은라이브키';
```

### 3단계: 서버 키 변경

**파일:** `Server/.env`

```bash
# 변경 전 (테스트)
TOSS_SECRET_KEY=test_sk_zXLkKEypNArWmo50nX3lmeaxYG5R

# 변경 후 (라이브)
TOSS_SECRET_KEY=live_sk_발급받은라이브시크릿키
```

### 4단계: 서버 재시작

```bash
cd Server
npm start
```

## 주의사항

⚠️ **라이브 키로 변경하면 실제 결제가 발생합니다!**

- 테스트 완료 후 변경하세요
- 변경 전에 모든 기능이 정상 작동하는지 확인하세요
- 실제 서비스 전환 시 SSL 인증서(HTTPS) 필수입니다

## 테스트 카드 정보 (테스트 모드에서만 사용 가능)

| 항목 | 값 |
|------|-----|
| 카드번호 | `4330-0000-0000-0001` |
| 유효기간 | 미래 날짜 (예: `12/30`) |
| CVC | 아무 3자리 (예: `123`) |
| 비밀번호 | 아무 2자리 (예: `12`) |

## 확인 사항

✅ 웹사이트에서 결제창이 정상적으로 열리는가?
✅ 결제 완료 후 라이선스 키가 생성되는가?
✅ 이메일로 라이선스 키가 발송되는가?
✅ 결제 취소 시 오류 메시지가 표시되는가?
